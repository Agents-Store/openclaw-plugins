# OpenClaw Plugins by Agents Store

Official OpenClaw plugins from [agents.store](https://agents.store)

## Plugins

| Plugin | npm | Description |
|--------|-----|-------------|
| openclaw-example | `@agents-store/openclaw-example` | Example plugin |

## Install

```bash
openclaw plugins install @agents-store/openclaw-example
```

## Development

```bash
pnpm install
pnpm build
```
